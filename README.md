# Работа с файлами

## Prerequisites

- **Python 3.13.7** или выше

Убедитесь, что Python установлен и добавлен в PATH. Проверить версию можно командой:

```bash
python3 --version
```

## Запуск

Для запуска приложения введите команду в терминале:

```bash
python3 contact_book_cli.py
```
