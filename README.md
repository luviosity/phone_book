# Phone Book – консольное приложение для управления контактами

Небольшое учебное приложение для хранения и управления контактами (имя, телефон и др.) в формате JSON через консольный интерфейс.

---

## Описание

Цель проекта – предоставить простой инструмент для работы с контактами в терминале, а также продемонстрировать базовую архитектуру приложения MVC.

Основные функции:

- Открытие JSON-файла (справочника)
- Добавление нового контакта
- Просмотр списка контактов
- Поиск контакта по имени, номеру телефона, комментарию или по всем полям
- Редактирование существующего контакта
- Удаление контакта
- Сохранение данных в JSON-файле

## Использование

1. Запустите основной файл приложения:

   - Вводите запрашиваемые данные.

## Структура проекта

Ключевые файлы и папки:

- `main.py`  
  Точка входа в приложение. Инициализирует основные компоненты и запускает цикл обработки пользовательских команд.

- `controller.py`  
  Связывает модель и представление. Обрабатывает пользовательский ввод, вызывает методы модели и передает данные во view.

- `model.py`  
  Логика работы с данными:
  - чтение и запись контактов в JSON-файлы;
  - реализаций операций со справочником.

- `view.py`  
  Отвечает за взаимодействие с пользователем:
  - вывод меню и сообщений;
  - чтение ввода из консоли;
  - отображение результатов операций.

- `custom_errors.py`  
  Пользовательские классы исключений, используемые в проекте для более точной обработки ошибок (например, контакт не найден, ошибка формата данных и т.п.).

- `custom_types.py`  
  Определения типов, структур данных или вспомогательные типы (например, тип контакта).

- `data.json`  
  Основной файл с данными телефонной книги. Хранит список контактов в формате JSON.

- `tools/`  
  Вспомогательные скрипты или утилиты. Например, классы для считывания записей из файла и записи данных в файл.
